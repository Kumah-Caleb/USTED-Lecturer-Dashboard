# USTED Lecturer Dashboard

Monitor student academic performance and identify at-risk students.

## Features
- View all students in your courses
- See latest predictions and risk levels
- Get alerts for at-risk students
- View analytics and grade distribution

## Deployment
This dashboard connects to the main USTED Predictor API.
